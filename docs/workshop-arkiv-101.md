# Arkiv 101 Workshop — May 15 @ Network School

A hands-on intro to building on Arkiv. **In-person at Network School** the day before the Builder Challenge kicks off.

> Material below is a stub. Slides, code samples, and the recording link will be filled in after the workshop runs. If you're remote, the X Spaces during May 1–14 cover the same ground at a higher level — see [README](../README.md).

---

## What you'll learn

By the end of the workshop, you'll have:

- A mental model for Arkiv's entity system (entities, attributes, relationships, expiration)
- A working dev environment connected to the Kaolin testnet
- Your first entity created, queried, and updated end-to-end
- A clear sense of which Builder Challenge theme you want to pursue

---

## Pre-requisites

- A laptop with Node.js 20+ installed
- A wallet (MetaMask / Rainbow / Frame — anything EVM)
- Basic JavaScript / TypeScript familiarity
- Some testnet ETH on Kaolin (faucet shared on the day)
